# Solidity Foundry Showcase

A portfolio repository containing multiple Solidity projects built with Foundry.

## Projects

### Crowdfunding
ETH crowdfunding contract with contribution tracking, withdrawals, and refunds.

### Token Staking System
Capped ERC20 token plus a staking contract with time-based rewards.

### MultiSig Treasury
Multi-owner treasury contract with transaction submission, confirmations, and execution.
